# Типы и структуры данных

Данный репозиторий содержит лабораторные работы, выполненные в рамках курса **«Типы и структуры данных»**.

Каждая лабораторная работа включает:

- исходный код на языке C++;
- отчёт, подготовленный в LaTeX;
- PDF-версию отчёта;
- дополнительные материалы, необходимые для выполнения и проверки работы.

## Используемые технологии

- C++
- LaTeX
- Алгоритмы и структуры данных

## Лабораторные работы

| Папка | Тема | Описание |
|--------|--------|--------|
| lab01 | Расстояние Левенштейна | Реализация алгоритмов Левенштейна и Дамерау–Левенштейна для вычисления редакционного расстояния между строками. |
| lab02 | Разреженные матрицы | Реализация структур хранения разреженных матриц и операций над ними. |
| lab03 | Стек и дек | Реализация структур данных стек и дек, а также основных операций над ними. |
| lab04 | Линейные списки | Реализация линейных списков и алгоритмов работы с динамическими структурами данных. |
| lab05 | Операции над матрицами | Реализация алгоритмов обработки и преобразования матриц. |
| lab06 | Задача коммивояжёра | Решение задачи поиска оптимального маршрута с использованием муравьиного алгоритма. |
| lab07 | Бинарные деревья | Реализация бинарных деревьев и алгоритмов обхода. |
| lab08 | Кодирование Хаффмана | Реализация алгоритма сжатия данных на основе кодов Хаффмана. |
| lab09 | AVL-деревья | Реализация самобалансирующегося бинарного дерева поиска AVL. |
| lab10 | Длинная троичная арифметика | Выполнение арифметических операций над длинными троичными числами. |

## Структура репозитория

Каждая лабораторная работа расположена в отдельной директории:

```text
lab01/
lab02/
...
lab10/
```

Каждая директория содержит:

- исходный код программы;
- исходный файл отчёта (`.tex`);
- PDF-версию отчёта;
- изображения и вспомогательные материалы.
